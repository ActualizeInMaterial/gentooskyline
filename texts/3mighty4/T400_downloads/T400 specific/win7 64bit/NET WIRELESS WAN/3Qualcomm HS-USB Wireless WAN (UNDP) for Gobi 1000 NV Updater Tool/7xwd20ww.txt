WHAT THIS PACKAGE DOES

  This package updates the Non-Volatile memory in the following device to fix
  problems, add new functions, or expand functions as noted below.

  Device name        Device names in the Device Manager
  ---------------    ----------------------------------
  Qualcomm HS-USB    Qualcomm HS-USB Modem 9202

  Refer to marketing materials to find out what computer models support which
  devices.

  This program is language independent and can be used with any language system.

HOW TO RUN THE SOFTWARE

Notes:
- If your computer runs satisfactorily now, it may not be necessary to update
  the software. To determine if you should update the software, refer to the
  Version Information section.
- The computer does not need to be logged on with an administrator account to
  run this software.
- The computer does not need to be rebooted after running this software.
- The device will be reset as part of this software's process.
- Confirm that the Qualcomm HS-USB device is powered on and operational.


Manual Install

  This section assumes to use Internet Explorer and Windows Explorer.

  Downloading file
  1. Click once on the underlined file name. Once this is done, some pop-up
     windows will appear.
  2. Follow the instructions on the screen.
  3. In the window to choose Run or Save, click Save.
  4. Choose the folder you would like to download the file to and click Save.
     A different window will appear and the download will begin and complete.
     Once the download has completed, there may or may not be a message stating
     that the download completed successfully.

  Extracting file
  5. Locate the folder where the file was downloaded.
  6. Locate the file that was downloaded and double-click it.
  7. Follow the instructions on the screen.
  8. In the Select Destination Location window, click Next. If you would like to
     select a different folder, click Browse.
  9. In the Ready to Install window, click Install. All the necessary files will
     be extracted to the folder selected in the step 9.

  Running the software
 10. Confirm that the Qualcomm HS-USB device is powered on and operational.
 11. Make sure the checkbox next to "Gobi NV Updater Tool" is checked and click
     Finish.
 12. Follow the instructions on the screen to complete installation and restart
     the computer.
     Or
     Locate the folder where the file was extracted the step 8, then double-
     click the Gobi1000Updater_DisableCB.exe file.

  Finally delete the file saved in the step 4.


Unattended Install

  There is no unattended installation for this software.

UNINSTALLATION INSTRUCTIONS


Manual Uninstall


 [Windows 7]

  1. Start Windows and logon with an administrator account.
  2. Click Start, then click Control Panel.
  3. Make sure "View by Category" is selected, and click Uninstall a program.
  4. Select version 1.0.00 (7XWD20WW), and click Uninstall or Uninstall/Change.
  5. Follow the instructions on the screen to complete uninstallation and
     restart the computer.


 [Windows Vista]

  1. Start Windows and logon with an administrator account.
  2. Click Start, then click Control Panel.
  3. Make sure "Control Panel Home" is selected, and click Uninstall a program.
  4. Select version 1.0.00 (7XWD20WW), and click Uninstall or Uninstall/Change.
  5. Follow the instructions on the screen to complete uninstallation and
     restart the computer.


 [Windows XP]

  1. Start Windows and logon with an administrator account.
  2. Click Start, then click Control Panel.
  3. Select Add or Remove Programs.
  4. Select version 1.0.00 (7XWD20WW), and click Remove or Change/Remove.
  5. Follow the instructions on the screen to complete uninstallation and
     restart the computer.
