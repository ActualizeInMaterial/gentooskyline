WHAT THIS PACKAGE DOES

  This package installs the software (SmartCard reader driver) to enable the
  following device.

  - Lenovo Integrated Smart Card Reader

  Refer to marketing materials to find out what computer models support
  which devices.

  This package also adds additional tabs to Device Manager.

  If this package has been installed, updating (overwrite-installing) this
  package will fix problems, add new functions, or expand functions as noted
  below.

  This program is language independent and can be used with any language system.

DETERMINING WHICH VERSION IS INSTALLED

  1. Click Start.
  2. Right-click Computer, and then click Properties.
  3. At the left pane, click Device Manager.
  4. Double-click the Smart Card Readers category.
  5. Double-click Integrated Smart Card Reader.
  6. Click the Driver tab.
  7. Check Driver Version.

INSTALLATION INSTRUCTIONS

Note:
- If your computer runs satisfactorily now, it may not be necessary to update
  the software. To determine if you should update the software, refer to the
  Version Information section.


Manual Install

  This section assumes to use Internet Explorer and Windows Explorer.

  Downloading file
  1. Click once on the underlined file name. Once this is done, some pop-up
     windows will appear.
  2. Follow the instructions on the screen.
  3. In the window to choose Run or Save, click the Save button.
  4. Choose the folder you would like to download the file to and click the Save
     button.
     A different window will appear and the download will begin and complete.
     Once the download has completed, there may or may not be a message stating
     that the download completed successfully.

  Extracting file
  5. Make sure to be logged on with administrative privileges.
  6. Locate the folder where the file was downloaded.
  7. Locate the file that was downloaded and double-click it.
  8. Follow the instructions on the screen.
  9. In the Select Destination Location window, click Next. If you would like to
     select a different folder, click Browse.
 10. In the Ready to Install window, click Install. All the necessary files will
     be extracted to the folder selected in the step 9.
 11. Uncheck the checkbox next to "Install ..... now", and then click Finish to
     complete extraction.

  Installing files
 12. Click Start.
 13. Right-click Computer, and then click Properties.
 14. At the left pane, click Device Manager.
 15. Double-click the Smart Card Readers category.
 16. Double-click Integrated Smart Card Reader or Microsoft Usbccid Smartcard
     Reader (WUDF).
 17. Click the Driver tab, then click Update Driver Software...
 18. Click Browse my computer for driver software.
 19. Click Browse..., then point to driver location specified in step 9.
 20. Make sure the checkbox next to "Include subfolders" is checked and click
     Next.
 21. Follow the instructions on the screen to complete installation and restart
     the computer.

  Finally delete the file saved in the step 4.

UNINSTALLATION INSTRUCTIONS


Manual Uninstall

  1. Start Windows 7 64-bit and log on with the user ID
     authorized as an administrator.
  2. Click Start, click Control Panel and open System and Security.
  3. Click System, then on the right side of the window, click Device Manager.
  4. Double-click on "Smart Card Readers" group to expand the category.
  5. Double-click on "Integrated Smart Card Reader".
  6. Select the Driver tab.
  7. Click on the <Uninstall> button.
  8. When prompted, confirm the operation by checking "Delete the driver software
     for this device" and then click <OK>.
  9. Uninstallation is now finished.


  Check the removal
  1. Start Windows 7 64-bit.
  2. Click Start, click Control Panel and open System and Security.
  3. Click System, then on the right side of the window, click Device Manager.
  4. Double-click on "SmartCardReader" group, verify that no "Integrated
     Smart Card Reader" appears any longer.
