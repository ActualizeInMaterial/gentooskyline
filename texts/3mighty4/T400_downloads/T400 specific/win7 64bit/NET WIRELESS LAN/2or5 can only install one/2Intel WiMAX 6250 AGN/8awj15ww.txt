WHAT THIS PACKAGE DOES

  This package installs the software (WiMAX driver and WiMAX function driver) to
  enable the following devices.

    Device name
    ------------------------------------------
    Intel Centrino Advanced-N + WiMAX 6250 AGN

    Device name in the Device Manager
    -----------------------------------------------
    Intel(R) Centrino(R) WiMAX 6250
    Intel(R) Centrino(R) WiMAX 6250 Function Driver

  Refer to marketing materials to find out what computer models support which
  devices.

  If this package has been installed, updating (overwrite-installing) this
  package will fix problems, add new functions, or expand functions as noted
  below.

  This program is language independent and can be used with any language system.

DETERMINING WHICH VERSION IS INSTALLED


Package:

 [Windows 8]

  1. Hold down the Windows logo key and press X to open a menu at the lower-left
     area of the screen.
  2. Select Programs and Features from the menu.
  3. Select Intel(R) PROSet/Wireless WiMAX Software.
  4. Check the product version in the Details Pane shown at the bottom of screen.
     If not shown, select Organize, select Layout, then select Details Pane.

 [Windows 7]

  1. Click Start, then click Control Panel.
  2. Make sure "View by Category" is selected, and click Uninstall a program.
  3. Click Organize, select Layout, then select Details Pane. The Details Pane
     will be shown at the bottom of screen.
  4. Select Intel(R) PROSet/Wireless WiMAX Software.
  5. Check the product version in the Details Pane.

 [Windows Vista]

  1. Click Start, then click Control Panel.
  2. Make sure "Control Panel Home" is selected, and click Uninstall a program.
  3. Click Organize, select Layout, then select Details Pane. The Details Pane
     will be shown at the bottom of screen.
  3. Select Intel(R) PROSet/Wireless WiMAX Software.
  5. Check the product version in the Details Pane.

 [Windows XP]

  1. Click Start, then click Control Panel.
  2. Select Add or Remove Programs.
  3. Select Intel(R) PROSet/Wireless WiMAX Software, then click "Click here for
     support information.".
  4. Check the software version.


WiMAX Driver:

 [Windows 8]

  1. Hold down the Windows logo key and press X to open a menu at the lower-left
     area of the screen.
  2. Select Device Manager from the menu.
  3. Double-click the Network adapters category.
  4. Double-click Intel(R) Centrino(R) WiMAX 6250.
  5. Click the Driver tab.
  6. Check Driver Version.

 [Windows 7/Vista]

  1. Click Start.
  2. Right-click Computer, and then click Properties.
  3. At the left pane, click Device Manager.
  4. Double-click the Network adapters category.
  5. Double-click Intel(R) Centrino(R) WiMAX 6250.
  6. Click the Driver tab.
  7. Check Driver Version.

 [Windows XP]

  1. Click Start.
  2. Right-click My Computer, and then click Properties.
  3. Click the Hardware tab, and then click Device Manager.
  4. Double-click the Network adapters category.
  5. Double-click Intel(R) Centrino(R) WiMAX 6250.
  6. Click the Driver tab.
  7. Check Driver Version.


WiMAX Function Driver:

 [Windows 8]

  1. Hold down the Windows logo key and press X to open a menu at the lower-left

  2. Select Device Manager from the menu.
  3. Double-click the Intel(R) Centrino(R) WiMAX Adapter category.
  4. Double-click Intel(R) Centrino(R) WiMAX 6250 Function Driver.
  5. Click the Driver tab.
  6. Check Driver Version.

 [Windows 7/Vista]

  1. Click Start.
  2. Right-click Computer, and then click Properties.

NOTES

- (For Windows XP only) Make sure that the following driver prerequisite were
  installed on your system before installing this driver:
  - Microsoft .NET Framework 3.0
    (Download installer from http://www.microsoft.com)

- UDP port 2948 needs to be opened in the firewall for successful activation and
  provisioning. Depending on the computer's firewall and security settings, the
  firewall's settings may need to be changed to allow the required
  communications.

- If you use ThinkVantage Access Connections, the following software must be
  installed.
  - Microsoft .NET Framework 3.0 or later (for Windows XP only)
  - ThinkVantage Access Connections
      version 5.90 or higher
  - ThinkPad Power Management Driver for Windows 98 SE/Me/2000/XP/Vista/7
      version 1.65 or higher
  - Hotkey Features Integration for Windows 7/Vista/XP
      version 4.04.0003 or higher.

- (For Windows 8 only) ThinkVantage Access Connect is not supported on Windows 8.
  Use Intel WiMAX Connections Technology utility found in the bottom right of
  the desktop task bar.

INSTALLATION INSTRUCTIONS

Note:
- If your computer runs satisfactorily now, it may not be necessary to update
  the software. To determine if you should update the software, refer to the
  Version Information section.


Manual Install

  This section assumes to use Internet Explorer and Windows Explorer.

  Downloading file
  1. Click once on the underlined file name. Once this is done, some pop-up
     windows will appear.
  2. Follow the instructions on the screen.
  3. In the window to choose Run or Save, click Save.
  4. Choose the folder you would like to download the file to and click Save.
     A different window will appear and the download will begin and complete.
     Once the download has completed, there may or may not be a message stating
     that the download completed successfully.

  Extracting file
  5. Make sure to be logged on with an administrator account.
  6. Locate the folder where the file was downloaded.
  7. Locate the file that was downloaded and double-click it.
  8. Follow the instructions on the screen.
  9. In the Select Destination Location window, click Next. If you would like to
     select a different folder, click Browse.
 10. In the Ready to Install window, click Install. All the necessary files will
     be extracted to the folder selected in the step 9.

  Installing files
 11. Make sure the checkbox next to "Install ..... now" is checked and click
     Finish.
 12. Follow the instructions on the screen to complete installation and restart
     the computer.

  Finally delete the file saved in the step 4.


Unattended Install

  This is for system administrators' use only.

  1. Refer to the Manual Install section, and download and extract the file.
  2. In the step 11 of the Installing files section, uncheck the checkbox next
     to "Install ..... now", and then click Finish to cancel installation.
  3. At the command line, execute AutoRun.exe with the /QN option.
     Example: [Path where the files were extracted]\AutoRun /QN

LIMITATIONS

  KNOWN ISSUES AND SOLUTIONS

- ThinkVantage Access Connections will not work if Windows XP Software Network
  Bridge connection is created.
  To recover the ThinkVantage Access Connections function:
  1. Start Windows XP and logon with administrative privileges.
  2. Click Start, select Settings, click Control Panel, then click Network
     Connections.
  3. Confirm that the following Device Name belongs to "Network Bridge".
       Intel(R) Centrino(R) WiMAX 6250
  4. Right-click its "Name" (Wireless Network Connection).
  5. Click Remove from Bridge.
  6. Confirm that it belongs to "LAN or High-Speed Internet".
  7. Right-click its "Name" again and click Properties.
  8. The Wireless Network connection Properties tab will open and under the
     General tab, Check at "WLAN Transport".
  9. Restart the system.
