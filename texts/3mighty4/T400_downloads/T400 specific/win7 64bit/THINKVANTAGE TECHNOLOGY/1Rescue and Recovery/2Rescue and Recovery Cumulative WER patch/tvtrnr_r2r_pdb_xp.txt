Rescue and Recovery Cumulative WER patch for Windows XP 


Introduction
============
This patch enables you to fix the critical WER issues of the ThinkVantage(R) Rescue and
Recovery(R) program on the Microsoft(R) Windows XP(R) operating system. After applying 
the patch, some Rescue and Recovery program files will be replaced and the performance of 
the related Rescue and Recovery services will be improved.

System requirements
===================
- Microsoft Windows XP operating system 
- Microsoft Internet Explorer(R) 5.5 or later 
- Rescue and Recovery 4.20 or 4.21 
- 1.5 GB of free hard disk drive space (required for installation)
- 512 MB of memory (recommended)
- 8 MB of VRAM (Video Random Access Memory) shared memory set in the BIOS
- Latest version of the BIOS 




Downloading and installing the patch
====================================
To download and install the patch, do the following:
1. Click the file link for the executable on the Web page.
2. When prompted, select a drive and directory to save the executable.
3. Go to the directory where the executable is saved.
4. Double-click the executable and follow the instructions on the screen to complete the installation.

Verifying the patch version
===========================
To verify the patch version, do the following:
1. Right-click the downloaded executable, and then click Properties.
2. Click the Details tab, and the version information will be displayed.
