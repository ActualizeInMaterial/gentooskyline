Patch to fix the compatibility issue between the Client Security Solution program 
and the Mozilla FireFox 3.6 Web browser

Installation Readme

Introduction

============

After you have installed the ThinkVantage(R) Client Security Solution program on 
your computer, the Password Manager plug-in appears dimmed and is disabled 
in the Mozilla Firefox 3.5 or 3.6 Web browser user interface.

This is a compatibility issue between the Client Security Solution program and 
the Mozilla Firefox 3.5 or 3.6 Web browser. This installation package provides 
a patch to help you solve this problem.

System requirements
===================

- Microsoft Windows(R) XP (32-bit), Windows Vista(R) (32-bit) or Windows 7 operating system
- With the installation of the Client Security Solution 8.21 (for the Windows XP or Windows Vista 
   operating system) or Client Security Solution 8.3 (for the Windows 7 operating system) program
- With the installation of the the Mozilla Firefox 3.5 or later Web browser 
- Latest version of the BIOS 
- 1.5 GB of free hard disk drive space 
- 512 MB of memory (recommended) 
- 8 MB of VRAM (Video Random Access Memory) shared memory set in the BIOS
- 1 GB of system memory for the hardware password reset NTFS (New Technology File System)

What you will need
==================

If you have an earlier version of the Client Security Solution program installed on the Windows XP or 
Windows Vista operating system, you need to upgrade the program to version 8.21 first. 
Then you can apply this patch.

To download the Client Security Solution 8.21 program for the Windows XP (32-bit) operating system, go to:

http://www-307.ibm.com/pc/support/site.wss/document.do?sitestyle=lenovo&lndocid=MIGR-72013

To download the Client Security Solution 8.21 program for the Windows Vista (32-bit) operating system, go to:

http://www-307.ibm.com/pc/support/site.wss/document.do?sitestyle=lenovo&lndocid=MIGR-72014

If you have installed the Windows 7 operating system, you can download and install this patch directly.

Downloading and installing the patch
=======================

1. Click the file link for the executable on the Web page.
2. When prompted, select a drive and directory to save the executable.
3. Go to the directory where the executable is saved.
4. Double-click the executable and follow the instructions on the screen to complete the installation.
