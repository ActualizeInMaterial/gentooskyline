WHAT THIS PACKAGE DOES

  This package installs the software (Bluetooth Software) to enable the
  following device.

  Device name              Device name in the Device Manager
  ---------------------    ---------------------------------------
  Realtek Bluetooth 4.0    Realtek Bluetooth 4.0 + High Speed Chip

  Refer to marketing materials to find out what computer models support which
  devices.

  If this package has been installed, updating (overwrite-installing) this
  package will fix problems, add new functions, or expand functions as noted
  below.

  This program is language independent and can be used with any language system.

DETERMINING WHICH VERSION IS INSTALLED


 [Windows 8 or later]

  [Mouse and Keyboard]
  1. Hold down the Windows logo key and press X to open a menu at the lower-left
     area of the screen.
  2. Select Device Manager from the menu.
  3. Double-click the Bluetooth Radios category.
  4. Double-click Realtek Bluetooth 4.0 + High Speed Chip.
  5. Select the Driver tab.
  6. Check Driver Version.

  [Touch]
  1. From the Start screen, select Desktop.
  2. Swipe in from the right edge of the touchscreen to display the charm bar.
  3. Select Settings.
  4. Select PC info.
  5. Select Device Manager.
  6. Double-tap the Bluetooth Radios category.
  7. Double-tap Realtek Bluetooth 4.0 + High Speed Chip.
  8. Select the Driver tab.
  9. Check Driver Version.


 [Windows 7]

  1. Select Start.
  2. Right-click/Press and hold Computer, and then select Properties.
  3. At the left pane, select Device Manager.
  4. Double-click/Double-tap the Bluetooth Radios category.
  5. Double-click/Double-tap Realtek Bluetooth 4.0 + High Speed Chip.
  6. Select the Driver tab.
  7. Check Driver Version.

INSTALLATION INSTRUCTIONS

Notes:
- If your computer runs satisfactorily now, it may not be necessary to update
  the software. To determine if you should update the software, refer to the
  Version Information section.
- If you update this software, uninstall the old version first before
  installation.


Manual Install

  This section assumes to use Internet Explorer and Windows Explorer.

  Downloading file
  1. Select the underlined file name. Once this is done, some pop-up windows
     will appear.
  2. Follow the instructions on the screen.
  3. In the window to choose Run or Save, select Save.
  4. Choose the folder you would like to download the file to, and select Save.
     A different window will appear and the download will begin and complete.
     Once the download has completed, there may or may not be a message stating
     that the download completed successfully.

  Extracting file
  5. Make sure to be logged on with an administrator account.
  6. Press Fn and F5 keys at the same time to turn on Bluetooth.
  7. If Found New Hardware Wizard is opened, click Cancel.
  8. Locate the folder where the file was downloaded.
  9. Locate the file that was downloaded and double-click/double-tap it.
 10. Follow the instructions on the screen.
 11. In the Select Destination Location window, select Next. If you would like
     to select a different folder, select Browse.
 12. In the Ready to Install window, select Install. All the necessary files
     will be extracted to the folder selected in the step 11.

  Installing files
 13. Close all applications.
 14. Make sure the checkbox next to "Install ..... now" is checked and select
     Finish.
 15. Follow the instructions on the screen to complete installation and restart
     the computer.
     Note: Make sure to restart the computer.

  Finally delete the file saved in the step 4.


Unattended Install

  This is for system administrators' use only.

  1. Refer to the Manual Install section, and download and extract the file.
  2. In the step 14 of the Installing files section, uncheck the checkbox next
     to "Install ..... now", and then select Finish to cancel installation.
  3. Close all applications.
  4. At the command line, execute setup.exe with the /s options.
     Example: [Path where the files were extracted]\setup /s
  5. When installation is completed, restart the computer.
     Note: Make sure to restart the computer.

UNINSTALLATION INSTRUCTIONS


Manual Uninstall


 [Windows 8 or later]

  [Mouse and Keyboard]
  1. Start Windows and logon with an administrator account.
  2. Hold down the Windows logo key and press X to open a menu at the lower-left
     area of the screen.
  3. Select Programs and Features from the menu.
  4. Select REALTEK Bluetooth Driver, and select Uninstall or Uninstall/Change.
  5. Follow the instructions on the screen to complete uninstallation and
     restart the computer.
     Note: Make sure to restart the computer.

  [Touch]
  1. Start Windows and logon with an administrator account.
  2. From the Start screen, select Desktop.
  3. Swipe in from the right edge of the touchscreen to display the charm bar.
  4. Select Settings.
  5. Select Control Panel.
  6. Make sure "View by Category" is selected, and select Uninstall a program.
  7. Select REALTEK Bluetooth Driver, and select Uninstall or Uninstall/Change.
  8. Follow the instructions on the screen to complete uninstallation and
     restart the computer.
     Note: Make sure to restart the computer.


 [Windows 7]

  1. Start Windows and logon with an administrator account.
  2. Select Start, then select Control Panel.
  3. Make sure "View by Category" is selected, and select Uninstall a program.
  4. Select REALTEK Bluetooth Driver, and select Uninstall or Uninstall/Change.
  5. Follow the instructions on the screen to complete uninstallation and
     restart the computer.
     Note: Make sure to restart the computer.
