WHAT THIS PACKAGE DOES

  This package installs the software (Bluetooth with Enhanced Data Rate Software)
  to enable the following devices.

  - ThinkPad Bluetooth with Enhanced Data Rate
  - ThinkPad Bluetooth 2.1 with Enhanced Data Rate

  However, some models do not support Bluetooth 2.1 due to a hardware
  restriction.

  Refer to marketing materials to find out what computer models support which
  devices.

  If this package has been installed, updating (overwrite-installing) this
  package will fix problems, add new functions, or expand functions as noted
  below.

  This program is language independent and can be used with any language system.

DETERMINING WHICH VERSION IS INSTALLED


 [Windows 7]

  1. Click Start, then click Control Panel.
  2. Make sure "View by Category" is selected, and click Uninstall a program.
  3. Click Organize, select Layout, then select Details Pane. The Details Pane
     will be shown at the bottom of screen.
  4. Select ThinkPad Bluetooth with Enhanced Data Rate Software.
  5. Check the product version in the Details Pane.


 [Windows Vista]

  1. Click Start, then click Control Panel.
  2. Make sure "Control Panel Home" is selected, and click Uninstall a program.
  3. Click Organize, select Layout, then select Details Pane. The Details Pane
     will be shown at the bottom of screen.
  4. Select ThinkPad Bluetooth with Enhanced Data Rate Software.
  5. Check the product version in the Details Pane.


  Note:
  This software is to add functions over the Microsoft Bluetooth software for
  Microsoft Windows Vista. So the version number of some of the software may not
  the same as the version number of this software.

INSTALLATION INSTRUCTIONS

Notes:
- If your computer runs satisfactorily now, it may not be necessary to update
  the software. To determine if you should update the software, refer to the
  Version Information section.
- If ThinkPad Bluetooth with Enhanced Data Rate Software is running on your
  ThinkPad computer, uninstall it before installing this version. Attempt to
  update installation and uninstallation from 6.2.0.7900 or previous to
  6.2.0.9600 or later will make explore to exit and restart. Icons, the Windows
  menu bar and task tray will disappear once and then back. This is a limitation
  of removing driver before 6.2.0.7900. This will not happen after 6.2.0.9600 to
  newer releases.


Manual Install

  This section assumes to use Internet Explorer and Windows Explorer.

  Downloading file
  1. Click once on the underlined file name. Once this is done, some pop-up
     windows will appear.
  2. Follow the instructions on the screen.
  3. In the window to choose Run or Save, click Save.
  4. Choose the folder you would like to download the file to and click Save.
     A different window will appear and the download will begin and complete.
     Once the download has completed, there may or may not be a message stating
     that the download completed successfully.

  Extracting file
  5. Make sure to be logged on with an administrator account.
  6. Close all applications.
  7. Press Fn and F5 keys at the same time to turn on Bluetooth.
  8. If Found New Hardware Wizard is opened, click Cancel.
  9. Locate the folder where the file was downloaded.
 10. Locate the file that was downloaded and double-click it.
 11. Follow the instructions on the screen.
 12. In the Select Destination Location window, click Next. If you would like to
     select a different folder, click Browse.
 13. In the Ready to Install window, click Install. All the necessary files will
     be extracted to the folder selected in the step 12.

  Installing files
 14. Make sure the checkbox next to "Install ..... now" is checked and click
     Finish.
 15. Follow the instructions on the screen to complete installation and restart
     the computer.
     Note: Make sure to restart the computer.

  Finally delete the file saved in the step 4.


Unattended Install

  This is for system administrators' use only.

  1. Refer to the Manual Install section, and download and extract the file.
  2. In the step 14 of the Installing files section, uncheck the checkbox next
     to "Install ..... now", and then click Finish to cancel installation.
  3. Close all applications.
  4. At the command line, execute setup.exe with the /qr /ri options.
     Example: [Path where the files were extracted]\Setup.exe /qr /ri
  5. Restart the computer.
     Note: Make sure to restart the computer.

UNINSTALLATION INSTRUCTIONS


Manual Uninstall


 [Windows 7]

  1. Start Windows and logon with an administrator account.

  4. Select ThinkPad Bluetooth with Enhanced Data Rate Software, and click
     Uninstall or Uninstall/Change.
  5. Follow the instructions on the screen to complete uninstallation and
     restart the computer.
     Note: Make sure to restart the computer.

  1. Start Windows and logon with an administrator account.
  2. Click Start, then click Control Panel.
  3. Make sure "Control Panel Home" is selected, and click Uninstall a program.
  4. Select ThinkPad Bluetooth with Enhanced Data Rate Software, and click
     Uninstall or Uninstall/Change.
  5. Follow the instructions on the screen to complete uninstallation and
     restart the computer.
     Note: Make sure to restart the computer.
