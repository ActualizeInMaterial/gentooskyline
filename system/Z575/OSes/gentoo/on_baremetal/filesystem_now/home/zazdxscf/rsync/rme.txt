Using these scripts from gentoo OS with username zazdxscf ( location: /home/zazdxscf/rsync/ folder)
to transfer files with rsync to/from the current gentoo OS into another folder
./gentoo/ then later copy this folder with relaxed permissions(all files
readable by zazdxscf) so that git can commit them into the gentooskyline repo.

Scripts also help with preparing a copy of the rsync-ed folder for storage inside git.
